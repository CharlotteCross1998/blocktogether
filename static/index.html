Block Together

An API app intended to help cope with harassers and abusers on Twitter. Users of Block Together can share their list of blocked users with friends, and can auto-block new users who at-reply them.

This is very much beta software. My hope is to provide something useful enough to be interesting, and build a community to help develop it. Everyone is welcome. Skills especially needed: Graphic design, UX design, HTML, CSS, and Javascript coding (the backend is in Node), and Ops.

Code is here. To get involved:

  • Visit the issues list and report a bug or request a feature.
  • Join the mailing list.
  • Tweet feedback and feature requests to @j4cob.
  • Submit a pull request!

This app will:
  • receive a list of the people you block
  • (optional) block additional people for you
  • (optional) share your block list with others
It will not tweet on your behalf. Note that Block Together needs read/write access to your account in order to block people when you ask it to. Twitter will describe that permission as 'Post tweets for you' on the permissions page.
